# 使用星星图标获取用户评分
Tags: meteor; bootstrap; glyphicon

------

使用一个数组，评分为3对应着：

html:

    {{#each rankicons}}
      <span class="glyphicon {{.}} text-danger"></span>
    {{/each}}

js:

    Template.UserEvaluations.helpers({
      rankicons: function() {
        return ['glyphicon-star', 'glyphicon-star', 'glyphicon-star', 'glyphicon-star-empty', 'glyphicon-star-empty'];
      }
    });

# Set Environment Variables in Windows
Tags: environment; windows;

------

Set env variables in current process: `set JAVA_HOME='D:\jdk8'`;
Set env vars globally: `setx JAVA_HOME 'D:\jdk8'`;
Print a env var: `echo %JAVA_HOME%`.

# TotalCMD 7.5常用快捷键
Tags: 快捷键; TotalCMD

------

wincmd.ini文件中的[Shortcuts]一节下面是默认和用户定义的快捷键，
可以通过修改这个文件设置TotalCmd的快捷键。

Ctrl + Up: 激活命令行，按esc关闭命令行；

shift + f4：新建一个文件 
shift + f12：同步文件夹 
alt + e：跳转到【我的电脑】 
alt + r：跳转到【网上邻居】，下面有【删除程序】等 
alt + w：跳转到【桌面】

alt + d: 编辑当前路径； 
：解压选定文件到当前目录下（alt+F9是解压到对面窗口） 

次常用快捷键： 
alt + a：文件夹是否也按时间排序
